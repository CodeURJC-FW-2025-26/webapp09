## Features:

### Main Entity
#### Product
#### Attributes
- ID: Unique product identifier
- Name: Product name
- Description: Product description
- Price: Product price
- Sizes: (S, M, L, XL)
- Category: (t-shirt, trouser, sneakers)

#### Secondary Entities
#### Review
#### Attributes
- review_id: unique identifier
- product_id: reference to the product
- user: user's name
- rating: product rating
